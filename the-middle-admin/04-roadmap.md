# Roadmap

## Phase 0: Admin Foundation

Goal: establish project memory and operating discipline.

Deliverables:

- admin charter
- project brief
- data model
- knowledge protocol
- inbox template
- decision log
- glossary

## Phase 1: Knowledge Base

Goal: create a searchable repository of documents, ideas, people, organizations, locations, cases, events, and concepts.

Deliverables:

- source registry
- document index
- entity registry
- concept library
- basic search

## Phase 2: Archive and Timeline

Goal: preserve records and make historical sequence visible.

Deliverables:

- document archive
- event timeline
- source confidence tracking
- version history

## Phase 3: Knowledge Graph

Goal: map relationships across people, organizations, documents, evidence, events, cases, and concepts.

Deliverables:

- graph schema
- relationship registry
- entity pages
- case relationship maps

## Phase 4: Intelligence Dashboard

Goal: turn structured knowledge into operational visibility.

Deliverables:

- case status dashboard
- organization registry
- risk view
- open questions view
- pending actions view

## Phase 5: AI Research Assistant

Goal: support deeper analysis while preserving human authority.

Deliverables:

- source summarization
- contradiction detection
- timeline extraction
- relationship suggestions
- decision support summaries

## Phase 6: Organizational Operating System

Goal: evolve The Middle into a complete knowledge and coordination platform.

Deliverables:

- permissions
- workflows
- review queues
- audit trail
- public/private knowledge views
- long-term institutional memory
