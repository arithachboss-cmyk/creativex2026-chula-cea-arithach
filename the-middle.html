AI Role

AI เป็นตัวกลางของข้อมูล ไม่ใช่ผู้ตัดสินความจริง

ระบบช่วยรวบรวม จัดหมวด เชื่อมโยง วิเคราะห์เบื้องต้น และเปิดช่องให้มนุษย์ตรวจสอบเสมอ

Roadmap

แผนพัฒนาจาก admin foundation ไปสู่ intelligence platform

Phase 0

Admin foundation: charter, brief, architecture, data model, protocol, registry, glossary.

Phase 1

Knowledge base: source registry, document index, entity registry, concept library, basic search.

Phase 2

Archive and timeline: preserve records, mark confidence, order events, track history.

Phase 3

Knowledge graph: relationship registry, entity pages, case maps, graph export.

Phase 4+

Intelligence dashboard, AI research assistant, workflows, permissions, audit trail.
