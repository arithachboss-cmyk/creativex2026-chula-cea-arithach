Tech Driven Creative Organization

Dashboard คลังความรู้ ดาวน์โหลด

Creative Organization

องค์กรสร้างสรรค์ต้องมีระบบที่ทำให้ไอเดียเดินทางได้เร็ว

สรุปแนวทางใช้ข้อมูล เทคโนโลยี และ workflow เพื่อช่วยให้องค์กรสร้างสรรค์เรียนรู้ ทำงานร่วมกัน และวัดผลได้จริง

Key Ideas

เทคโนโลยีที่ดีไม่ใช่เพียงเครื่องมือ แต่เป็นโครงสร้างที่ทำให้ความรู้ถูกเก็บ ค้นหา นำไปใช้ และต่อยอดเป็นผลงานได้

Operating Model

ควรมีฐานข้อมูลความรู้ ระบบสมาชิก สิทธิ์การเข้าถึง Dashboard และวงจร review เพื่อให้ผลงานมีคุณภาพก่อนเผยแพร่

Learning Questions

ข้อมูลใดควรถูกเก็บเป็นฐานกลาง ขั้นตอนใดควรอัตโนมัติ และผู้บริหารต้องเห็นตัวชี้วัดใดเพื่อสนับสนุนทีมได้ตรงจุด
