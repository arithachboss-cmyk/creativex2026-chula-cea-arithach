# Kamakaben Scent Learning And Rehab Framework

## Purpose

ออกแบบกระบวนการเรียนรู้อัตลักษณ์วัฒนธรรมไทยผ่านกลิ่น เพื่อให้ผู้ใช้ทั่วโลกไม่เพียง “ชอบกลิ่น” แต่เข้าใจว่าแต่ละกลิ่นพาไปสัมผัสสถานที่ ความทรงจำ และภูมิปัญญาไทยอย่างไร

## Learning Model

### 1. Sense

ให้ผู้ใช้ดมก่อนอ่าน ให้ร่างกายและความทรงจำตอบสนองก่อนข้อมูล

Prompt:

What did this scent make you remember before you knew its story?

### 2. Name

ให้ผู้ใช้ตั้งชื่ออารมณ์ สถานที่ หรือภาพที่เกิดขึ้น

Prompt:

If this scent were a place in Thailand, where would it be?

### 3. Decode

เปิดข้อมูลวัฒนธรรม วัตถุดิบ ภูมิประเทศ พิธีกรรม หรือเรื่องเล่าที่อยู่หลังกลิ่น

Prompt:

Which Thai material, ritual, or place does this scent introduce?

### 4. Ritualize

ให้ผู้ใช้ใช้กลิ่นซ้ำใน ritual ที่ชัด เช่น ก่อนนอน ก่อนทำงาน หลังเดินทาง หลังเรียนรู้

Prompt:

When in your day does this scent help you return to yourself?

### 5. Reflect

ให้ผู้ใช้บันทึกความรู้สึก ความทรงจำ และสิ่งที่เรียนรู้

Prompt:

What did this scent teach you about Thailand, and what did it teach you about yourself?

### 6. Share

เปลี่ยน reflection เป็น social proof, review, memory card, workshop output หรือ community archive

Prompt:

What story would you share with someone who has never been to Thailand?

## Experience Formats

### Scent Card

- Scent name
- Notes
- Thai cultural anchor
- Memory prompt
- Ritual instruction
- QR to knowledge page

### Workshop

- Blind smell
- Memory writing
- Cultural decode
- Group conversation
- Personal ritual design
- Take-home scent card

### Digital Journey

- Scan QR
- Listen to audio story
- Read cultural note
- Save personal reflection
- Share scent memory

## Safety And Ethics

- ใช้คำว่า rehab ในเชิง emotional reset, cultural reconnection, learning recovery
- ห้ามอ้างรักษาโรคหรือผลทางการแพทย์
- ให้เครดิตชุมชน วัตถุดิบ ภูมิภาค และผู้ร่วมสร้างสรรค์
- ระวังการใช้ภาพไทยแบบซ้ำซากหรือทำให้วัฒนธรรมดูเป็นของตกแต่ง
