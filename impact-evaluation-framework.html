Evaluation Framework

ประเมินผลจากกิจกรรม สู่ความสามารถ ผลงาน และผลกระทบ

ใช้กรอบนี้เพื่อออกแบบตัวชี้วัด เก็บหลักฐาน และรายงานผลลัพธ์ของแพลตฟอร์มต่อผู้บริหารและผู้สนับสนุน

1. Inputs

งบประมาณ บุคลากร วิทยากร พันธมิตร ข้อมูลกลาง ระบบสมาชิก และพื้นที่อัปโหลดความรู้

2. Activities

การเรียนรู้ การค้นหา การดาวน์โหลด การส่งผลงาน การอัปโหลดไฟล์ การรีวิว และการให้ feedback

3. Outputs

จำนวนบทความ ไฟล์ดาวน์โหลด สมาชิก active ผลงานที่ส่ง กิจกรรมคะแนน และคำค้นหายอดนิยม

4. Outcomes

ความรู้ที่เพิ่มขึ้น ความร่วมมือใหม่ ผลงานที่ใช้ต่อได้ และการตัดสินใจเชิงนโยบายที่ดีขึ้น

5. Impact

การยกระดับเศรษฐกิจสร้างสรรค์ ความสามารถในการแข่งขัน ภาพลักษณ์เมือง และคุณภาพของระบบนิเวศสร้างสรรค์
